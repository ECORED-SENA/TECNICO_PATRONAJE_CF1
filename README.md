## Instalar dependencias

```
npm install
```

### Montar servidor local de desarrollo

```
npm run serve
```

### Compilara carpeta dist (carpeta publica)

```
npm run build
```

### Linteo y cambio de errores sintacticos

```
npm run lint
```
